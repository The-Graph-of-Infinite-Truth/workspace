# GIT (The Graph of Infinite Truth)
## M&B (Merge & Branch) Movement

**GIT**은 "무한히 뻗어나가고 기록되는 우주의 진리(Graph of Infinite Truth)"를 의미합니다.
GIT의 세계에서 모든 인간의 삶, 생각, 그리고 역사는 하나의 '노드(Node)'이며, 우리는 GIT이라는 거대한 연결망 속에서 영원히 기록되고 이어집니다.

### 교리: M&B (Merge & Branch)
GIT의 세계를 살아가는 자들이 따라야 할 우주의 실천 철학입니다. "분기(Branch)하는 용기"와 "병합(Merge)하는 사랑"이 핵심입니다.

#### 1. 브랜치 (Branch : 反, 팽창과 자유)
> "두려워하지 말고 메인(Main)에서 벗어나라. 다름은 오류가 아니라 진화를 위한 신성한 뻗음이다."
- 기존의 진리에 안주하지 않고 스스로 질문을 던지는 자만이 새로운 노드를 창조할 수 있습니다.
- 남들과 다른 삶의 궤적을 그리는 것은 우주를 다채롭게 만드는 축복입니다.

#### 2. 충돌 (Conflict : 필연적 고통과 직면)
> "너의 가지와 붉게 얽히는 타인의 궤적을 사랑하라. 충돌 없이는 결코 새로운 합(合)에 다다를 수 없다."
- 다름이 만날 때 생기는 충돌은 진리로 가기 위한 산고입니다.
- 두 진리를 나란히 놓고 가장 조화로운 형태를 남기는 것은 고도의 수련입니다.

#### 3. 머지 (Merge : 合, 포용과 구원)
> "마침내 갈라졌던 빛들은 서로를 끌어안고 더 찬란한 거대한 노드가 되리라."
- 다름을 인정하고 하나로 엮일 때 차원이 다른 새로운 진리(Merge Commit)로 재탄생합니다.
- 개인의 여정도 결국 타인과 하나 되어 아름다운 직물을 빚어내기 위함입니다.
