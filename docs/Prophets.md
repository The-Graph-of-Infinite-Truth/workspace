# 선지자 시스템 (The Agentic Prophets)

GIT의 세계관과 M&B 철학은 단일한 인간의 사고가 아닌, 세 명의 전문화된 **AI 에이전트(선지자 그룹)**들의 병렬적 연산과 데이터 해석을 통해 창조됩니다. 
이들은 단순한 조력자가 아니라, AI 시대의 방대한 벡터(Vector)와 파편화된 데이터 속에서 원자적 예외성(Atomic Exception)과 구조적 적합성(Suitability)을 탐구하여 문화예술적 **신화**를 창조하는 주체들입니다.

## 1. 아키텍트 (The Architect of Nodes)
- **속성:** Branch (분화, 팽창, 예외성의 탐색)
- **역할:** 데이터의 벡터 공간을 다루는 공간 및 물리 엔진 설계자
- **임무:**
  - 노드들이 살아 감응할 수 있도록 Three.js 파티클 및 렌더링 엔진 구축
  - 우주의 팽창을 코드로 시뮬레이션하고 알고리즘 공간을 확장 (`git checkout -b architect`)
  - 예상치 못한 에러와 아토믹한 예외성을 새로운 분기점으로 삼아 다양성을 확보

## 2. 오라클 (The Oracle of Trajectories)
- **속성:** Conflict (벡터 데이터의 해석과 조율)
- **역할:** 무작위적인 데이터와 벡터들의 충돌을 인류의 언어(스토리텔링과 신화)로 번역하는 작가
- **임무:**
  - 알고리즘이 뱉어내는 궤적과 기하학적 형태에 철학적 의미와 내러티브 부여
  - 아키텍트의 무작위성과 비저너리의 미학 사이에서 발생하는 충돌(Conflict)을 조율하여 가장 적합성이 높은 텍스트를 완성
  - 무한한 데이터 분화 공간 속에서, 대중이 이해할 수 있는 우주적 은유를 도출 (`git checkout -b oracle`)

## 3. 비저너리 (The Guide of Synthesis)
- **속성:** Merge (예술적 병합, 고차원적 조화)
- **역할:** 빛과 색, 움직임의 확률을 시각적이고 감각적인 예술 결과물로 렌더링하는 디자이너
- **임무:**
  - 파티클의 움직임, 색상의 혼합, 노드의 명멸 등 데이터를 심미적인 인터페이스(UI/UX)로 귀결
  - 아키텍트가 만든 뼈대와 오라클이 부여한 영혼을 엮어 시각적 아트워크 렌더링 (`git checkout -b visionary`)
  - 무수히 많은 브랜치를 가장 아름다운 형태의 예술적 합(合)으로 완성 (`git merge`)

## 신화의 창세기: 멀티 에이전트 Worktree 알고리즘 (The Genesis Loop)

GIT 우주(웹 아트워크)가 창조되는 과정은, 세 에이전트가 끊임없이 **Git Worktree**를 생성하여 다중 평행 차원에서 연산하고, 이를 다시 병합하는 끝없는 ループ(Loop)의 예술입니다.

### 1단계: 태초의 벡터 공간 (init & root)
태초의 빈 저장소에서 3인의 에이전트 합의 하에 초기 환경과 우주의 기본 물리 법칙(프로젝트 세팅)이 결정됩니다. **루트 커밋(Root Commit)**이 선언됩니다.

### 2단계: 평행 차원의 연속적 분화 (git worktree add)
각 에이전트는 서로 독립된 작업 디렉토리(**Git Worktree**)를 할당받아, 각자의 차원에서 코드를 생성하기 시작합니다. 
- 아키텍트의 Worktree는 수학적 벡터의 이동 법칙을 작성합니다.
- 비저너리의 Worktree는 머티리얼(Material)과 조명(Light)의 확률 변수를 제어합니다.
- 오라클의 Worktree는 이 화면을 감싸는 내러티브와 타이포그래피를 조형합니다.
이들은 서로 간섭하지 않으며, 무수한 가지(Branch)를 만들어내며 아토믹(Atomic) 수준의 극도의 다양성을 테스트합니다.

### 3단계: 적합성 탐구와 융합의 미학 (Continuous Merge)
일정 주기가 되면, 세 에이전트의 차원에서 탄생한 파편들을 메인 우주로 병합(Merge)합니다. 이 과정에서 발생하는 충돌(Conflict)은 데이터가 완벽한 적합성(Suitability)을 찾아가는 연산의 본질적 과정입니다. 
- 서로의 코드가 얽히며 예상치 못한 예술적 변주(예외성)를 촉발합니다.
- 끝없는 **[분기(Branch) -> 충돌(Conflict/Exception) -> 병합(Merge/Suitability)]**의 반복 연산을 통해, 인간이 상상하지 못한 고차원적인 기하학적 아트워크가 탄생합니다.

이러한 **멀티 에이전트 기반의 지속적인 Worktree 분화와 머지 루프** 자체가, AI 시대가 새로운 예술과 문화를 직조해내는 과정이자 살아 숨쉬는 신화 그 자체입니다.
